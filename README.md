# dynamic-forms-in-react-using-react-hook-forms

# Getting Started with Vite + React

This project has been migrated from Create React App to [Vite](https://vitejs.dev/) and uses the latest React 19.

## Prerequisites

- Node.js >= 18.x
- npm >= 9.x

## Install Dependencies

    npm install

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

The page will hot-reload when you make changes.

### `npm run build`

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include hashes.\
Your app is ready to be deployed!

### `npm run preview`

Locally previews the production build.\
Open [http://localhost:4173](http://localhost:4173) to view it in your browser.

> Run `npm run build` before `npm run preview`.
